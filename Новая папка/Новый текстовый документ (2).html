Контрольные вопросы:


4.1 Почему в браузере не отображается текст, расположенный между тегами ?
Ответ: Текст, расположенный между тегами , является комментарием и не должен отображаться в браузере. Комментарии используются для вставки заметок и объяснений в код HTML, которые не должны быть видимы для пользователей.

4.2 Какое утверждение наиболее подходит к случаю, когда тег <р> располагается в контейнере ?
Ответ: Тег <р> не должен располагаться в контейнере , так как содержит метаданные, необходимые для корректной работы страницы, а не контент для отображения.

4.3 Дайте определение элементу main. Какова его цель?
Ответ: Элемент <мain> является контейнером для основного контента веб-страницы. Его цель заключается в том, чтобы выделить основную часть содержимого документа, которая может быть уникальной для каждой страницы. Это помогает поисковым системам и ассистентам пользователя лучше понимать структуру страницы и улучшает доступность для пользователей с ограниченными возможностями.

4.4 Какой функцией можно изменить размер ячейки?
Ответ: Для изменения размера ячейки таблицы можно использовать свойство CSS "width" или "height".

4.5 Чем отличается div от span?
Ответ: Основное отличие между div и span заключается в том, что div используется для создания блочных элементов, а span - для создания строчных элементов.

4.6 Как обозначаются комментарии в html?
Ответ: Комментарии в HTML обозначаются с помощью тега . Любой текст, заключенный между этими тегами, будет проигнорирован браузером при отображении страницы.

4.7 При помощи какого атрибута можно избавиться от зазоров между ячейками таблцы?
Ответ: Для избавления от зазоров между ячейками таблицы в HTML используется атрибут cellspacing.
